Welcome to Wallander, a data inspector.

Data Provider: is a Python object that provides access to one or more Data Sources.
Data Source: is a Python object that contains one or more fields.
Data Field: the type of data provided by a data source.

URLs

data provider URL: 
/wallander/[data_provider]

data URL: 
/wallander/[data_provider]/[data_source_path].../[data_source]/[data_field]{/[slice]}{/[function]{/[function]{...}}
image URL: 
/frames/[data_provider]/[data_source_path].../[data_source_field]-{kwargs}-[frame]

slice format

See Numpy Indexing

function format

Each Data Provider may control the functions they supply. The Base Data Provider exposes all the attributes of the NumPy module that return true when passed to inspect.isfunction.

Colormaps

name-min-max-over-under-bad
over,under,bad formated as hex strings RRGGBBAA. Any name can be appended with _log to get a log color map.

Examples

A colormap that uses Purple to Blue base colormap, has a min/max of 1e19/1e24 and an under color of #2DD6D6 with and alpha of 0.2 PuBu_log-1e19-1e24-2DD6D633
